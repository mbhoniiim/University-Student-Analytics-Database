University Student Analytics Database

Project Overview

The University Student Analytics Database is a PostgreSQL-based data analytics project designed to store and analyse student academic performance data across different faculties, modules, and lecturers.

The goal of this project is to demonstrate database design, SQL querying, and business intelligence skills by answering key academic performance questions using real-world analytical techniques.



Objectives

This project aims to:

- Design a normalized relational database.
- Store student, module, lecturer, and marks data.
- Analyse academic performance using SQL.
- Generate insights that can support university decision-making.
- Serve as a portfolio project for Data Analyst and BI Analyst roles.



Technologies Used

- PostgreSQL
- pgAdmin 4
- SQL
- Mockaroo (Data Generation)
- GitHub



Database Structure

The database consists of five tables:

Faculties
Stores information about university faculties.

2. Students
Stores student information.

3. Lecturers
Stores lecturer information.

4. Modules
Stores module information.

5. Marks
Stores student marks.


Business Questions Answered

1. Top Performing Students

Identify students with the highest average marks.

Business Value:
- Scholarship allocation
- Academic awards
- Student recognition


2. Pass Rates Per Module

Calculate the percentage of students passing each module.

Business Value:
- Measure module success rates
- Identify underperforming courses


3. Average Marks By Faculty

Compare academic performance across faculties.

Business Value:
- Faculty benchmarking
- Resource allocation


4. Most Difficult Modules

Identify modules with the lowest average marks.

Business Value:
- Curriculum review
- Academic support planning



Sample Insights

Examples of insights this database can provide:

- Which students consistently achieve the highest marks?
- Which modules have the lowest pass rates?
- Which faculty performs best academically?
- Which modules may require intervention or additional support?
